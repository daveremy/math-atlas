# Research Plan — Atlas v0.1

1. **Taxonomy pass**: MSC 2020 top levels → candidate territory list (~12). Compare against Princeton Companion's organization. Decide the atlas's own top level (neither slavishly).
2. **Core-concept harvest**: per territory, the 15–30 concepts a serious generalist map must contain — sourced from Companion chapter structure, standard curricula, Metacademy graphs.
3. **Edge pass**: prerequisite/dependency edges, marked by kind (hard prereq / soft support / illuminates).
4. **Leverage pass**: centrality computation over the edge set; mathlib-derived empirical weights where mappable; curated adjustment with rationale per override.
5. **Two-face stubs**: one-line intuition gloss + one-line formal identity per node (full content later).
6. **Render**: infinite-canvas interactive viewer (Stage-2 tool of the parent project's CONCEPT.md).
7. **Review**: adversarial pass (codex + independent reviewer) on structure and claims before v0.1 is called done.

Cadence: territory-by-territory, sessions logged; or accelerated via multi-agent workflow on request.
